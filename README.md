## 🚀 Getting Started

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn

### Installation

1. **Clone the repository**
   
bash
   git clone https://github.com/Satyam1042716/portfolio.git
   cd portfolio


2. **Install dependencies**
   
bash
   npm install


3. **Start development server**
   
bash
   npm start


4. **Open in browser**
   
http://localhost:300


## 📦 Build for Production

bash
npm run build
